# LinkedIn Automation

A Chrome extension designed to automate interactions on the LinkedIn Feed, specifically liking and commenting on posts.

## Project Structure

- **extension/**: Chrome extension for automating feed interactions.

## Prerequisites

- [Google Chrome](https://www.google.com/chrome/)

## Setup

### Chrome Extension

1.  Open Google Chrome and navigate to `chrome://extensions`.
2.  Enable **Developer mode** in the top right corner.
3.  Click **Load unpacked** in the top left corner.
4.  Select the `extension` directory from this project.

## Usage

1.  Ensure you are logged into LinkedIn in your Chrome browser.
2.  Click on the extension icon to open the popup.
3.  Enter the number of posts to "Like" and "Comment".
4.  Click "Start Automation".
5.  The extension will open the LinkedIn Feed, scroll, and perform the actions.
